# Key Commands

A list of the commands and handoffs used when developing Claude harness.

## Key Commands (used during development)

| Command | Purpose |
|---------|------|
| `/plan-with-agent` | Add improvement tasks to Plans.md |
| `/work` | Implement tasks (auto-scope detection) |
| `/breezing` | Full team parallel run with Agent Teams |
| `/reload-plugins` | Apply skill/hook edits immediately (no restart needed) |
| `/harness-review` | Review changes |
| `/validate` | Validate plugin |
| `/remember` | Record learnings |

## Handoff

**Skills (auto-invoked in conversation)**:
- `handoff-to-impl` - "hand off to the implementer" → PM → Impl request
- `handoff-to-pm` - "report completion to PM" → Impl → PM completion report
